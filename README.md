## Required modules

| Package | Command (Windows) |
| ------ | ------ |
| PyOpenGL | py -m pip install PyOpenGL glfw |
| numpy | py -m pip install numpy |
| PIL | py -m pip install Pillow |
| PyGLM | py -m pip install PyGLM |
